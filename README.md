# homebrew-tap

Homebrew formulae for [iilei](https://github.com/iilei) projects.

## Usage

```sh
brew install iilei/tap/gitsnitch
brew install iilei/tap/proquint
```

## Formulae

### gitsnitch

A Git commit history linter written in Rust.

**Source:** [github.com/iilei/gitsnitch](https://github.com/iilei/gitsnitch)

```sh
brew install gitsnitch
```

After installation, verify with:

```sh
gitsnitch --version
```

### proquint

A command-line utility for Proquint encoding and decoding.

**Source:** [github.com/iilei/proquint](https://github.com/iilei/proquint)

```sh
brew install iilei/tap/proquint
```

After installation, verify with:

```sh
proquint --version
```
